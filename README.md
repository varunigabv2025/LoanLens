# LoanLens

LoanLens is an educational web application that helps first-time borrowers understand loans, EMIs, repayment schedules, common lending terms, and scam alerts.

## Features
- EMI Calculator
- Repayment Schedule
- Loan Dashboard
- Financial Terms Guide
- Scam Alert Awareness
- Offline Functionality

## Technologies Used
- HTML
- CSS
- JavaScript
